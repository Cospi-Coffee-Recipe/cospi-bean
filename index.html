Cospi

개인정보 처리방침

Cospi는 스페셜티 커피 레시피, 추출 기록, 친구 공유 기능을 제공하기 위해 필요한 범위에서 개인정보를 처리합니다.

시행일: 2026년 6월 13일

1. 처리하는 개인정보 항목

Cospi는 회원 기능과 데이터 동기화를 제공하기 위해 아래 정보를 처리할 수 있습니다.

로그인 및 계정 Firebase 사용자 ID, 이메일 주소, 표시 이름, 프로필 사진 URL, 로그인 제공자 정보, 익명 로그인 여부, 가입일 및 갱신일
소셜 로그인 Google, Apple, Kakao 로그인 과정에서 제공되는 인증 정보 및 프로필 정보. Kakao 액세스 토큰은 로그인 확인과 Firebase 인증 토큰 발급 목적으로만 사용됩니다.
친구 기능 친구 코드, 친구의 사용자 ID, 표시 이름, 프로필 사진 URL, 이메일 주소, 친구 추가 시각
레시피 및 추출 기록 사용자가 만든 커스텀 레시피, 즐겨찾기와 조회 정보, 추출 세션, 평점, 산미, 단맛, 바디, 쓴맛, 태그, 메모, 공유 설정, 작성자 정보
서비스 이용 과정 앱 오류, 접속 로그, IP 주소, 기기 및 브라우저 정보 등 서비스 운영과 보안에 필요한 기술 정보가 Firebase, Google Cloud, GitHub Pages 등에서 생성될 수 있습니다.

2. 개인정보의 처리 목적

  • 회원 식별, 로그인 유지, 계정 관리
  • 사용자 레시피, 즐겨찾기, 추출 기록 저장 및 여러 기기 간 동기화
  • 친구 추가, 친구가 공유한 레시피와 추출 기록 표시
  • 공식 레시피와 원두 정보 제공, 서비스 품질 개선
  • 부정 이용 방지, 보안, 장애 대응, 고객 문의 처리

3. 개인정보의 보관 및 삭제

Cospi는 서비스 제공에 필요한 기간 동안 개인정보를 보관합니다. 사용자가 직접 삭제한 레시피, 추출 기록, 친구 정보는 서비스 데이터에서 삭제됩니다. 계정 삭제 또는 개인정보 삭제를 요청하면 본인 확인 후 Firebase Authentication 계정과 Firestore에 저장된 사용자 데이터를 삭제합니다.

관계 법령에 따라 보관이 필요한 정보가 있는 경우에는 해당 법령에서 정한 기간 동안 분리 보관할 수 있습니다.

4. 개인정보의 제3자 제공 및 처리 위탁

Cospi는 사용자의 개인정보를 판매하지 않습니다. 다만 서비스 운영을 위해 아래 서비스가 개인정보를 처리할 수 있습니다.

Google Firebase 및 Google Cloud 인증, 데이터 저장, 클라우드 함수 실행, 서비스 운영 로그 처리
Google, Apple, Kakao 사용자가 선택한 소셜 로그인 인증 및 프로필 정보 제공
GitHub Pages 개인정보 처리방침 페이지 호스팅 및 접속 로그 처리

친구 공유 기능을 켠 레시피와 추출 기록은 친구로 등록된 사용자에게 표시될 수 있습니다.

5. 국외 이전

Cospi는 Firebase, Google Cloud, GitHub Pages 등 글로벌 클라우드 서비스를 사용합니다. 이에 따라 개인정보 또는 서비스 운영 로그가 서비스 제공자의 국외 인프라에서 저장되거나 처리될 수 있습니다. 이전되는 정보는 서비스 제공, 보안, 장애 대응 목적에 필요한 범위로 제한됩니다.

6. 이용자의 권리

이용자는 언제든지 개인정보 열람, 정정, 삭제, 처리 정지를 요청할 수 있습니다.

  • 앱 안에서 직접 삭제할 수 있는 레시피, 추출 기록, 친구 정보는 해당 기능을 통해 삭제할 수 있습니다.
  • 계정 삭제, 전체 데이터 삭제, 기타 개인정보 관련 요청은 계정 및 데이터 삭제 요청 안내를 확인하거나 아래 문의 이메일로 연락해 주세요.

7. 안전성 확보 조치

  • Firebase Authentication을 통한 사용자 인증
  • Firestore 보안 규칙을 통한 사용자별 데이터 접근 제한
  • HTTPS 기반 통신과 클라우드 서비스의 보안 기능 사용
  • 운영 목적상 필요한 인원과 범위로 접근 권한 제한

8. 아동의 개인정보

Cospi는 만 14세 미만 아동을 주 대상으로 하지 않으며, 만 14세 미만 아동의 개인정보를 고의로 수집하지 않습니다. 법정대리인이 아동의 개인정보 삭제를 요청하는 경우 확인 후 필요한 조치를 진행합니다.
